== Description ==

This plugin helps you do quality control on your new WordPress site. As you come across a page that needs to be corrected the enclosed bookmarklet will create a dialog box on the page. Enter a to do item, select the to do list and then it is added to Basecamp.

== Installation ==

1. Upload the plugin to the plugins folder.
2. Goto the Treu Quality Control Menu under the Settings menu.
3. Add your Basecamp subdomain, API key and company id.
4. Drag the bookmarklet link at the top of the settings page to your address bar.
5. Start adding items to your Basecamp to do list.
